PROTECT LIFE SCIENCES – FULL WEBSITE DEVELOPMENT PLAN

---

1. PROJECT OVERVIEW

---

Goal:
Build a professional B2B pharmaceutical website for Protect Life Sciences.

Core Focus:

* Third-party manufacturing (main USP)
* Export business (Africa + global)
* Product showcase (antibiotics, injections)
* Lead generation (contact form + email)

Tech Stack:

* Frontend: React + Vite + TypeScript + Tailwind CSS
* Routing: React Router
* Form Handling: EmailJS (initial)
* Deployment: Vercel
* Domain: GoDaddy (protectlifesciences.in)

15. DEPLOYMENT

---

Frontend:

* Deploy on Vercel

Steps:

1. Push code to GitHub
2. Connect repo to Vercel
3. Deploy

Domain Setup (GoDaddy):

1. Go to DNS settings
2. Add Vercel nameservers or A record
3. Link domain in Vercel
